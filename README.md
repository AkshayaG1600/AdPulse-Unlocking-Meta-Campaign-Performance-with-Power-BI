# AdPulse-Unlocking-Meta-Campaign-Performance-with-Power-BI
This project presents an interactive Ad Performance Dashboard built using Power BI to analyze advertising campaigns on Facebook and Instagram. It converts raw campaign data into actionable insights, enabling marketing teams to make data-driven decisions and improve campaign ROI.

**🎯 Aim of the Project**

The main objective of this project is to:

- Develop an interactive Power BI dashboard for ad performance monitoring
- Provide insights into reach, engagement, conversions, and budget utilization
- Analyze campaign effectiveness across platforms (Facebook vs Instagram)
- Enable data-driven decision-making to improve ROI and campaign efficiency

**📈 Dashboard Features & Visualizations**

The dashboard includes multiple interactive visuals:

🟠 Donut Chart – Performance by Gender
📊 Bar Chart – Engagement by Age Group
🌍 Map Visualization – Country-wise performance
📅 Calendar Heatmap – Monthly activity trends
📊 Stacked Column Chart – Weekly performance by ad type
📉 Area Chart – Hourly engagement trends
📋 Matrix Table – Ad type vs platform comparison

**Key Business Questions Answered**

🔹 Performance & Effectiveness

- Which platform performs better (Facebook vs Instagram)?
- Which ad type drives the highest engagement and conversions?
- How effective are ads across the conversion funnel?

🔹 Audience Insights

- Which gender engages the most?
- What is the most responsive age group?
- Which countries contribute the most engagement?

🔹 Time-Based Insights
- What are the peak days and months?
- What time of day drives maximum engagement?

**🛠️ Tools & Technologies Used**

- Power BI – Data visualization & dashboard development
- Data Modeling – Relationship building & KPI calculations
- DAX – Measures and calculated fields
